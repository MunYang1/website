My second heading where I insert an image

Picture is shown by &img src="URL LINK" alt="NAME" style="width=128px;height;128px;">. I will use <br> to break the sentence abrup
ly
src is the source of the image, where an URL link of the image is shown, I did this by copying the link of the image from discord (an online messenger)
alt will provide a name for the picture, in the case that the image link doesn't work, it will display an empty image as well as the name set in the alt to describe to the viewer what the image was meant to be.
style will read from the style section that was set at the top of this html, which will keep to it's original resolution
